## 🔧 Setup Instructions

1. Clone this repo:
   ```bash
   git clone https://github.com/manoharvallabi/Portfolio.git
   cd Portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open in browser:  
   👉 `http://localhost:5173`

## 🚀 Deployment

This portfolio is deployed on [Vercel](https://vercel.com/).

To deploy your own version:
1. Push the project to your GitHub repo.
2. Go to [vercel.com](https://vercel.com/), log in with GitHub, and import your repo.
3. Vercel will auto-detect the project (React + Vite) and deploy it automatically.

Live site: [https://manoharvallabi.vercel.app](https://manoharvallabi.vercel.app) *(update if custom domain is different)*

## 🛠️ Customize Your Info

Update the content in `src/content/`:
- `personal.json` – Name, title, bio, social links
- `projects.json` – Your portfolio projects
- `skills.json` – Your tech stack
- `experience.json` – Work and education history
